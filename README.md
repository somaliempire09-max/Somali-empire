# 🏴 Somali Empire Discord Bot

Discord bot leh laba nidaam oo weyn:
- 🌸 Welcome Setup Panel
- 🏴 Gang Wars (Territory, Missions, Leaderboard)

## Setup

```bash
npm install
```

`.env` file samee:
```
DISCORD_TOKEN=your_token_here
```

Bot bilow:
```bash
npm start
```

## Commands (Prefix: `?`)

| Command | Sharaxaad |
|---|---|
| `?help` | Dhammaan commands |
| `?welcome` | Welcome Setup Panel (Admin) |
| `?gang create <name>` | Gang cusub samee |
| `?gang join <name>` | Gang ku biir |
| `?gang leave` | Gang ka tag |
| `?gang info` | Gang macluumaad |
| `?gang members` | Xubnaha arag |
| `?gang bank` | Lacagta arag |
| `?gang upgrade` | Upgrade-yada |
| `?gang promote @user` | Xil kor u qaad |
| `?gang demote @user` | Xil hoos u dhig |
| `?gang mission list` | Missions arag |
| `?gang mission start <id>` | Mission bilaw |
| `?gang mission join` | Mission ku biir |
| `?gang mission status` | Mission hubi |
| `?territory` | Dhulalka arag |
| `?attack <territory>` | Dhul weerari |
| `?defend <territory>` | Dhul difaaci |
| `?leaderboard` | Tartanka arag |
| `?startwar` | War bilaw (Admin) |
| `?stopwar` | War jooji (Admin) |
